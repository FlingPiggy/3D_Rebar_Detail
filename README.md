# 3D Rebar Detail

Browser-based 3D schematic tool for visualising reinforcement in concrete members.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Usage

- **Rotate**: left-click drag
- **Pan**: right-click drag  
- **Zoom**: scroll wheel
- **View presets**: Iso / Top / Front / Side buttons (bottom-left of viewport)
- **Save**: exports `rebar-model.json`
- **Open**: loads a previously saved JSON file
- **Copy Code**: copies a compressed base64 shortcode to clipboard
- **Load Code**: paste a shortcode to restore a model
- **Example**: loads the built-in sample column model
